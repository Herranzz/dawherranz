Cómo agregar Iconos

Para insertar un icono, agregue el nombre de la clase de icono a cualquier elemento HTML en línea. Los elementos < i> y < span> son ampliamente utilizados para agregar iconos. Todos los íconos en las bibliotecas de íconos a continuación son íconos vectoriales escalables que se pueden personalizar con CSS (tamaño, color, sombra, etc.)
